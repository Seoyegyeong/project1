You can use this program with command 'make'
and then when you enter './main', the program is starting.
	e.g) make -> ./main

There are 12 choice including exit choice.
1) Create : create new restaurant info.
2) Read list : read all records.
3) Update : modify one record.
  (I'll show you restaurant name list and you have to pick one number, then modify)
4) Delete : delete one record.
  (restaurant name list -> pick one number)
5) Read one : read one record, not all.
  (restaurant name list -> pick one number)


6) Search(menu) : give you a list about restaurants which sells what are you searching. (검색한 메뉴를 파는 식당 이름과 전화번호를 출력해줍니다.)
7) Search(delivery) : give you a list about restaurants which have delivery service (배달 가능한 식당을 알려줍니다.)
8) Load : 파일에 있는 식당정보를 메모리에 가져옵니다.
9) Save : 현재 메모리에 있는 식당정보를 텍스트 파일로 옮겨 담아줍니다.

10) add dibs : 원하는 식당을 찜합니다.(좋아요 버튼 누르는 식)
   (restaurant name list -> pick one number)
11) dibs list : 찜해둔 식당의 리스트를 출력해줍니다.

if you want to stop, you can enter 0 or !(1~11).

Enjoy this!
